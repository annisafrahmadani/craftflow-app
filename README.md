# 🚀 CraftFlow

CraftFlow adalah aplikasi web untuk membantu pengguna mengelola project, task, dan job application dalam satu dashboard.

## ✨ Features

- Dashboard
- Project Management
- Task Management
- Job Tracker
- Authentication
- Protected Routes
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React
- React Router
- CSS
- Chart.js

### Backend
- Django
- Django REST Framework

## 📂 Project Structure

```
craftflow-app/
│
├── backend/
├── frontend/
└── README.md
```

## 🚀 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
